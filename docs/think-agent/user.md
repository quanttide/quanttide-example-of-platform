意图澄清工具 — 用户指南

这是什么？

意图澄清工具是一个帮助你理清思路、对齐认知的AI协作工具。

在日常工作中，当我们面对一个新问题、新方向或新需求时，往往一开始并不清楚具体要做什么。需要反复讨论、尝试、推翻、再尝试，才能逐渐形成清晰的方案。这个工具就是为这个“从模糊到清晰”的探索阶段设计的。

它不写正式代码，不管需求流转，只做一件事：帮你在探索过程中，和AI一起维护一份“我们当前对这件事的共同理解”。

—

解决什么问题？

你是否有过这样的经历：

· 和AI聊了很久，但发现它做的事和你想要的不太一样，反复纠正很累
· 探索了好几轮，回头翻聊天记录，很难快速回忆起“我们试过什么、结论是什么”
· 终于想清楚了，但要整理成一份可用的文档或BRD，还得手动重写一遍

这个工具就是为解决这些问题而设计的。

—

核心概念：意图模型

工具围绕一个叫意图模型的东西工作。你可以把它理解为“我和AI当前对这件事的共同理解”，它包含四个方面：

维度 含义 示例
目标 我要达成什么 “找到提升用户留存的核心切入点”
当前探索 我正在尝试什么方向 “假设新用户引导流程太复杂是流失主因，正在验证”
约束 有什么限制条件 “不能动后端接口；上线时间两周内”
状态 目前到哪一步了 “已分析流失漏斗，确认引导流程第三步断崖明显”

这四个维度的内容，就是你和AI在探索过程中共同的认知基线。

—

怎么用？

第一步：开始对话

像平时聊天一样，在左侧对话框告诉AI你想探索什么。比如：

“我们在考虑提升用户留存，但还没想清楚从哪入手。”

AI会跟你讨论，同时自动检测你的意图变化。

第二步：AI自动更新意图模型

当你和AI对话中出现关键转折时——比如你提出了一个新方向、推翻了一个旧假设、或者明确了一个新约束——AI会自动更新右侧的意图模型面板，并在对话中提示你。

例如，AI可能会提示：“意图已更新：探索方向从‘引导流程优化’转向‘推送时机调整’”。

第三步：你随时校准

右侧意图模型面板的每一栏都可以直接点击编辑。如果你觉得AI理解得不对，或者你想修改某个维度的内容，直接点进去改就行了。改完AI立刻会按照新的理解继续工作。

第四步：验证你的想法（可选）

如果你需要快速验证某个想法是否可行，工具可以调用编程能力生成临时的验证代码并执行，返回结果。注意：这些代码只是为了验证想法而临时生成的，不会保留。验证结果会帮你进一步更新意图模型。

第五步：导出为文档

当你觉得已经想清楚了，点击导出按钮。意图模型的四个部分会自动拼接成一份结构化的文档，可以直接作为BRD、调研记录或下一步工作的输入，复制到其他地方使用。

—

典型使用场景

场景一：方向探索

“我想探索一下我们产品的增长机会在哪。”
你和AI从不同角度讨论，可能先聊渠道，再聊产品功能，最后聚焦到某个具体假设。整个过程中，意图模型会持续记录你们从发散到收敛的认知轨迹。

场景二：需求澄清

“业务方提了一个大方向的需求，但细节很模糊。”
你把已知信息告诉AI，在对话中逐步澄清目标、边界和约束。导出后的文档可以直接作为和业务方对齐的依据。

场景三：问题诊断

“某个指标突然下降，不知道原因。”
你提出一个又一个假设，AI帮你逐个分析，并在意图模型中记录哪些被验证、哪些被排除、当前最可能的原因是什么。

—

和普通聊天工具的区别

普通聊天工具 意图澄清工具
聊完就过去了，需要手动翻记录回顾 意图模型持续外化，随时可查看当前共识
AI可能越聊越偏，需要反复纠正 AI自动检测意图变化，并始终以模型为基准
聊完了要手动整理文档 导出即文档，无需二次整理

—

快速上手建议

1. 开始时不用刻意维护意图模型，正常对话即可。AI会自动判断什么时候该更新。
2. 每当你感觉“方向变了”或“有新结论”的时候，看一眼右侧面板，确认AI理解是否准确。
3. 不准确就直接改。你是最终决策者。
4. 想清楚了就导出，不一定要等到“完美”——意图模型就是帮你更快达到“足够清楚”的状态。
