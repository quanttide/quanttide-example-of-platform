# asset-agent 构建系统

## 愿景

构建一套多智能体协作质量管理系统，让管理者同时看清分工、过程和成本三个维度。

## 系统架构

```
                    ┌──────────────┐
                    │   事件总线    │
                    └──────┬───────┘
                           │
              ┌────────────┼────────────┐
              ▼            ▼            ▼
      ┌──────────┐ ┌──────────┐ ┌──────────┐
      │ 版本追踪  │ │ 审计日志  │ │ 边界配置  │
      └────┬─────┘ └────┬─────┘ └────┬─────┘
           │             │            │
           ▼             ▼            ▼
      ┌───────────────────────────────────┐
      │          渲染层                   │
      │  ┌─────────┐ ┌─────────┐ ┌──────┐ │
      │  │ 结构树   │ │ 演化图   │ │审计表│ │
      │  │(分工)   │ │(过程)   │ │(成本)│ │
      │  └─────────┘ └─────────┘ └──────┘ │
      └───────────────────────────────────┘
```

## 当前焦点

两个原型各自验证了核心设计点，下一步是统一事件数据源，使版本追踪和审计日志共用同一套事件总线。
